    快速选中内容添加括号¶

    <esc>->v 进入visual mod, 然后移动以选中

    c 将选中内容剪切

    ()输入括号

    <ctrl + c>->p 粘贴

    批量注释¶

    <esc>->v / <ctrl + v> / <shift + v> 进入visual mod

    <shift + i> 多行添加

    输入批量添加的信息

    <esc>-><esc> 退出
